A modern web-based **Canteen Management System** built using **HTML, CSS, JavaScript, and Firebase**.  
This application streamlines food ordering, menu management, and role-based access for Students, Staff, and Admin users.

---

##  Features

###  Student Panel
- User Registration & Login
- Browse available food items
- Place orders in real-time
- View order status updates

###  Staff Panel
- View incoming orders
- Update order status (Pending / Preparing / Completed)
- Toggle menu item availability

###  Admin Panel
- Add / Edit / Remove menu items
- Manage pricing & availability
- Monitor system activity

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend / Database:** Firebase Realtime Database
- **Authentication:** Firebase Authentication
- **Hosting (Optional):** Firebase Hosting / Netlify
- **Development Environment:** VS Code
